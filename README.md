# Vue 3 + TypeScript + Vite


一个集成的一站式预览插件，能支持常见项目场景中的需求，对word、execl、pptx、pdf、各类文本、图片，音视频的预览，
并且要求具备以下特点：

- 开箱即用，使用简单便捷
- 一站式，同一环境下支持多类型多格式
- 兼容各主流浏览器、适配PC端/移动端/T信，统一风格
- 提供必要配置项、多文件资源格式输入，更加灵活通用
- 支持统一导入和按需引入
- 基于 vitepress 构建的文档，较好的预览体验
